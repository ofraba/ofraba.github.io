תיאור הקורס


מהפכת האינטרנט שינתה את פני העולם והטכנולוגיה. יישומים נוספים עולים לרשת מידי יום ביומו <>על מנת לתת מענה להתפתחות זו, יש להכשיר אנשי מקצוע מיומנים ובעלי יכולות פיתוח הנותנות מענה הן בצד השרת והן בצד הלקוח,וכתוצאה מכך נוצר תפקיד ייעודי בתחום הפיתוח הידוע כ-Full stack developer.
מפתחי Full stack שולטים במגוון עצום של יכולות טכנולוגיות הן בצד השרת והן בצד הלקוח ומאפשרים בנייה אינטראקטיבית והצגה דינמית של מידע ותוכן, ומאידך, הבנה עמוקה של תהליכי הפיתוח הנדרשים על מנת לעבד את הנתונים בצד השרת ולהנגישם לצד לקוח.
חברות פיתוח בסביבת האינטרנט זקוקות לפיתוח דינמי ומידי כדי לתת מענה לפיתוח צד שרת וצד לקוח. היחידה ללימודי חוץ של הטכניון כיחידה מובילה בתחומה, כתבה את התכנים בהתאם לצרכי השוק. מסלול זה מוכר כ- Full stack developer בסביבת הקוד הפתוח וכולל בתוכו את טכנולוגיות ה-MEAN וה-MERN.
המסלול מיועד למי שמעוניין להכיר וללמוד את היכולות של מרבית הטכנולוגיות המוכרות בעולם פיתוח אתרי האינטרנט הן בצד הלקוח והן בצד השרת. הקורס חושף את הסטודנט לעולם הפיתוח של הקוד הפתוח המבוסס על סביבת Java Script יחד עם פיתוח קלאסי של בסיס נתונים רלאציוני וסביבת פיתוח קלאסית כמו NET.
התכנית שמה דגש על תרגול מעשי רב, לרבות ביצוע 3 פרויקטים אשר ישקפו את הידע הנרכש בקורס וכחלק בלתי נפרד מהמסלול יפתח התלמיד אתר אינטרנט שלם בעצמו.
מטרת התכנית הכרות לעולם פיתוח היישומים באינטרנט הכשרת מתכנתים לפיתוח צד לקוח הכולל טכנולוגיות כגון HTML5, CSS3, Java Script, Angular, React הכשרת מתכנתים לפיתוח צד שרת הכולל טכנולוגיות כגון NodeJS, .NET, Firebase הכשרת מתכנתים לעבודה מול בסיסי נתונים וארכיטקטורות בסיסי נתונים
הכשרת מתכנתים Full Stack על מנת לתת מענה מקצועי וטכנולוגי תוך הבנת התהליכים יתרונות התכנית התכנית מעניקה לבוגריה את היכולת להיקלט ולהשתלב במגוון חברות פיתוח בתחום האינטרנט עם היכולות לתת מענה הן לפיתוח בצד שרת והן לפתוח בצד לקוח
התוכנית מספקת הבנה מעמיקה של תהליכים וארכיטקטורות המבצעות שימוש בטכנולוגיות ה-WEB שיטת הלימוד מבוססת כולה על תרגול וצבירת ניסיון אשר יכין את הבוגר אותו לאתגרים בתעשייה
המסלול פותח ע"י צוות מקצועי, המורכב מטובי המומחים בתחום פיתוח בסביבת הקוד הפתוח ובעלי ניסיון מקצועי רב הן בהדרכה והן בפיתוח בתעשייה התוכנית שמה דגש על לימוד מעמיק של 2 טכנולוגיות ה- Web המובילות כיום בתעשייה: הן Angular ו- React
(יתרון אדיר בראיונות עבודה מול מועמדים אחרים) הבוגרים מבצעים במהלך התוכנית 3 פרויקטים
